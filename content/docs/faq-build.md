### Comment écrire des commentaires en JSX ? {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* Placer un commentaire ici */}
  Bonjour, {name} !
</div>
```

```jsx
<div>
  {/* Ça fonctionne aussi pour
   les commentaires sur plusieurs lignes */}
  Bonjour, {name} !
</div>
```
